## Purpose

This app was created to:

- Track match scores between friends.
- Display a leaderboard.
- Have a quick and fun way to see who's winning (or losing).

> [!NOTE]
> This is not a production-ready app. It's a fun side project with minimal features and no serious development intentions.

## Tech Stack

- **Next.js**: For the frontend and backend logic.
- **TypeScript**: To keep things somewhat organized.
- **Neon**: For the database (PostgreSQL).

## Features

- **Player Management**: Add and manage players.
- **Match Creation**: Start matches between players.
- **Score Tracking**: Enter match results and update scores.
- **Leaderboard**: View the current rankings.
- **Stats Page**: View detailed statistics about matches and players.

## Disclaimer

This project was built in a few hours for fun. It's not meant to be a serious or polished application. Expect bugs, security flaws, and missing features. Use at your own risk!
